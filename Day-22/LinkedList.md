# Linked List 

* Linear data structure
* nodes are connected together by a referece field 

**Example 
const linedList = {
    point : 1
    node : {
        point : 2,
        node : {
            point : 3,
            node : null
        }
    }
}

# Type of Linked List : 
 * 1. Singly Linked List
 * 2. Doubly Linked List
